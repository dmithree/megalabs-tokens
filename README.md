## Releasing

1. Run `git checkout master && git pull` to get the latest changes.
2. Add the new version number to the `CHANGELOG.md`.
3. Commit the changes on `master` with `git add CHANGELOG.md && git commit -m "Preparing for release."`
4. Run `yarn release`.
